# FoodOrderingApp

Create a food Ordering App using some pre-set data {json}. The JSON can look like :

[{"name":"ABC",
  "id":"123", 
  "location":"testLocation",
  "rating":"4.2",
  "tags":["healthy","snacks","Indian"],
  "eta":"25",
  "photo":"<url to the pic>",
  "distance": "10kms"
  }]
  Features developed:
  1. Rendering the restaurants in cards on UI
  2. Search functionality
  3. Sorting based on name, rating , distance and ETA
  4. Filter based on the tags.
  5. Marking any restaurant as fav
